# **App Name**: WaveLink

## Core Features:

- Secure Login: Secure merchant registration and login.
- Payment Request Creation: Create payment requests including the amount, customer name, WhatsApp number, and product description.
- Wave Payment Link Generation: Generate a unique Wave payment link for each transaction using the merchant's Wave account details.
- Automatic WhatsApp Notification: Automatically send a structured order confirmation message to the customer via WhatsApp, including the Wave payment link, using Wasender API.
- Payment Status Tracking: Track the real-time status of payments (paid, pending, failed) and automatically update the status.
- Transaction History: View a history of transactions, with clear tracking of status and customer information.
- Intelligent Commission Tool: Based on previous history of the merchant, intelligently decides between charging a commission per transaction, or offering a flat monthly subscription plan to the merchant.

## Style Guidelines:

- Primary color: Deep blue (#293B5F) to convey trust and security in financial transactions.
- Background color: Light gray (#F0F4F8), a desaturated version of the primary hue for a clean and professional look.
- Accent color: Turquoise (#3D5A80), an analogous color that provides a contrasting, yet harmonious, element to guide user actions.
- Headline font: 'Space Grotesk', a sans-serif font for headlines and short amounts of body text, creating a tech-forward, clear impression; body text will use 'Inter'
- Body font: 'Inter', a grotesque-style sans-serif for a neutral, objective feel.
- Simple, clean icons to represent payment status, transaction types, and user actions.
- Subtle transitions and loading animations to enhance user experience during payment processing and data updates.
